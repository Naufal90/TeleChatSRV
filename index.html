TeleChatSRV

Plugin Minecraft x Telegram terbaik untuk monitoring server real-time

Apa Itu TeleChatSRV?

TeleChatSRV adalah plugin Spigot/Bukkit/Paper yang menghubungkan server Minecraft Anda dengan Bot Telegram. Cocok untuk admin server yang ingin memantau aktivitas pemain secara jarak jauh, termasuk notifikasi pemain, pembunuhan, dan deteksi penggunaan xray.

Fitur Utama

  • Notifikasi Join/Quit: Pemain masuk dan keluar dari server.
  • Notifikasi Pembunuhan: Pesan saat pemain membunuh atau terbunuh.
  • Anti-Xray Deteksi: Deteksi dan notifikasi saat pemain menghancurkan block berharga.
  • Integrasi Telegram Mudah: Cukup isi token dan chat ID.

Contoh Notifikasi Telegram

  • [JOIN] Naufal90 baru saja masuk ke server.
  • [KILL] Naufal90 membunuh Steve.
  • [XRAY?] Naufal90 menghancurkan DIAMOND_ORE di X:120 Y:11 Z:-45.

Cara Konfigurasi

  1. Buat bot Telegram dengan @BotFather.
  2. Dapatkan token dan masukkan ke dalam file config.yml.
  3. Dapatkan chat ID menggunakan bot @userinfobot.
  4. Sesuaikan block yang ingin dipantau melalui block_notify_filter.

Contoh config.yml

bot_token: "ISI_TOKEN_BOT_ANDA"
chat_id: "ISI_CHAT_ID_ANDA"block_notify_filter: DIAMOND_ORE: true GOLD_ORE: true IRON_ORE: false
